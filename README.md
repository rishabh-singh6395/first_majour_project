# Wanderlust 🌍

### A Full-Stack Airbnb Inspired Travel Accommodation Platform Built From Scratch

Wanderlust is a full-stack web application inspired by the core workflow of Airbnb. The purpose of this project was to understand and implement real-world web development concepts by building a complete application from scratch.

Instead of focusing on creating a completely new idea, this project focuses on mastering important development concepts such as authentication, database management, backend architecture, CRUD operations, image handling, and building a scalable application structure.

---

## 🚀 Features

### 🔐 User Authentication

* User signup and login functionality
* Secure authentication using sessions
* Authorization system for protected actions

### 🏡 Property Listings

* Users can create their own property listings
* Edit and delete listings
* Add important details:

  * Title
  * Description
  * Location
  * Price
  * Images

### ⭐ Reviews & Ratings

* Users can review properties
* Add ratings
* Delete their own reviews
* Connected user-listing-review relationship

### 🖼️ Image Management

* Upload property images
* Store and manage images using cloud storage

### 📱 Responsive Design

* User-friendly interface
* Responsive layout for different screen sizes

# 🏗️ Application Architecture

Wanderlust follows the MVC (Model-View-Controller) architecture:

```
Wanderlust
│
├── Models
│   └── Database schemas and relationships
│
├── Views
│   └── User interface templates
│
├── Controllers
│   └── Application logic
│
├── Routes
│   └── API and page routing
│
└── Middleware
    └── Authentication and validations
```
